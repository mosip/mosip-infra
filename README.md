# MOSIP Infra

## Overview
We provide a reference implementation of a Kubernetes-based, **production grade** deployment of MOSIP, also referred to as the **V3 deployment**. This implementation supports both **sandbox** environments for evaluation and testing, as well as **full-scale production deployments**.

The installation process is divided into two distinct phases, which are detailed [here](deployment/v3/README.md).
## Documentation

This repository contains various scripts and procedures to deploy [MOSIP](https://docs.mosip.io/1.2.0/setup/deploymentnew/v3-installation/mosip-modules-deployment) in sandbox and production environments.
