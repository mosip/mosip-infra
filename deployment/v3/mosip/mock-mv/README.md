# Mock MV

## Introduction
Mock MV is the mock implementation of manual adjudication or verification system. Mock MV is a microservice installed inside MOSIP K8 cluster.

## Install
* To install mock-mv.
```sh
./install.sh
```
## Restart
* To restart mock-mv.
```sh
./restart.sh
```
## Delete
* To delete mock-mv.
```sh
./delete.sh
```
