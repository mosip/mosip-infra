# Mock ABIS

## Introduction
Mock ABIS is a simulation of a real ABIS where interface with ActiveMQ is simulated. Mock ABIS runs inside the cluster as a Kubernetes pod.

## Install 
* To install mock-abis.
```sh
./install.sh
```
## Restart
* To restart mock-abis.
```sh
./restart.sh
```
## Delete
* To delete mock-abis.
```sh
./delete.sh
```
